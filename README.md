# -Sonic-vs.-Eggman-Iron-Odyssey-PC-Edition-
PC port of Sonic vs. Eggman: Iron Odyssey, optimized for high-integrity systems. Built on Sovereign-Gateway-Core using Rust, C++, and Solidity to bridge legacy engines with modern decentralized environments. Led by Richard A. DiMassa, this project integrates agentic workflows for low-latency, "Zero-Stall" performance.
